# Optics

This page introduces the main Optics concepts used throughout the project.

## What is Optics?

Optics provides a structured way to model composable behaviors and transformations.

## Core idea

- Keep APIs small and focused.
- Compose operations safely.
- Use clear naming for transformations and data flow.
