# Composition

Composition is the process of building higher-level behavior from smaller reusable parts.

## Why compose

Composition reduces duplication and makes behavior easier to reason about.

## Typical pattern

1. Define small, single-purpose units.
2. Combine units in a clear order.
3. Validate outputs at each layer.
