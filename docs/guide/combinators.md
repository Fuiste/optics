# Combinators

Combinators are building blocks used to combine or transform values and operations in a consistent manner.

## Basic guidance

- Prefer explicit, readable combinations over clever one-liners.
- Keep combinator inputs and outputs obvious.
- Document expected behavior for each combinator.

## Suggested naming

- `map`
- `filter`
- `reduce`
- `chain`
- `compose`
