# Contributing and editing docs

## Contributing and editing docs

Use short, lowercase markdown filenames in `docs/wiki` and keep links local.

### Edit process

1. Add or update the page under `docs/wiki`.
2. Keep section headings consistent for anchor stability.
3. Update `docs/wiki/index.md` hub links when adding/removing pages.
4. Mirror link style from the conventions page.

### Quality checklist

- New pages use `.md` extension.
- Internal links use `./` or `../` only.
- Anchor references match exact heading slugs.
