# API glossary

## API glossary

- `Lens`: Focuses on required values and supports reading/writing.
- `Prism`: Focuses on optional/discriminated values.
- `Iso`: Bi-directional, invertible conversion.
- `Traversal`: Focuses multiple values.
- `Getter`: Read-only focused getter.
- `Fold`: Read-only multi-read focus.
- `compose`: Core function for combining optics.
- `guard`: Helper for type-guard based `Prism` construction.
- `at`: Key-based `Prism` for record access.
- `index`: Index-based `Prism` for array access.
- `each`: Traversal over array members.
