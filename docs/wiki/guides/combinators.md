# Combinators guide

## Combinators guide

Use combinators to build derived optics from common patterns.

### `guard`

Create a partial optic from a predicate or user-defined type guard.

### `at`

Create a record-key prism that is no-op safe when values are missing.

### `index`

Create a numeric index prism for array-like structures.

### `each`

Create a traversal over array elements.
